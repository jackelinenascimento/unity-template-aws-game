## Como usar

1. Abra o arquivo do prompt no Kiro.
2. Copie o conteúdo e cole no chat do Kiro.
3. Aguarde o Kiro ler o projeto e implementar.
4. **Teste no Unity antes de pedir o próximo prompt.**

> Regra de ouro: uma alteração de cada vez → testa → só então pede a próxima.
