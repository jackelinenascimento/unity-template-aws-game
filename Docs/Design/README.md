# 🎨 Docs / Design — Ctrl+Alt+Delas

Decisões de design, referências visuais e rascunhos do jogo.
Use esta pasta para registrar o que o time decidiu durante o evento — serve como guia para quem pega o bastão na rotação de papéis.

---

## O que colocar aqui

- **Tema escolhido** — qual o universo visual do jogo (laboratório sci-fi, floresta, cidade, espaço…)
- **Paleta de cores** — cores principais que os assets devem seguir
- **Referências visuais** — prints de jogos ou imagens que inspiraram o design
- **Personalizações decididas** — quais das 4+ personalizações o time vai fazer
- **Mecânicas extras** — se o time decidiu adicionar algo além do mínimo, documente aqui
- **Rascunhos de fase** — esboços do layout (pode ser uma foto de papel)

---

## 📝 Template de decisões do time

Crie um arquivo `decisoes.md` aqui com este formato:

```markdown
# Decisões de Design — Ctrl+Alt+Delas

## Tema
[descreva o universo visual aqui]

## Paleta de cores
- Cor primária: #______
- Cor secundária: #______
- Cor de destaque: #______

## Personalizações escolhidas
- [ ] Novo personagem
- [ ] Novo cenário
- [ ] Novos obstáculos
- [ ] Novos itens
- [ ] Nova mecânica: ________________
- [ ] Nova expressão OLED: ________________

## Layout da fase
[descreva ou cole um esboço]

## Mecânicas extras
[descreva qualquer mecânica além do mínimo]
```

---

## 💡 Dica para o time

Decidir o tema **antes** de gerar os assets evita que cada membro da equipe gere coisas em estilos incompatíveis. 5 minutos de alinhamento no começo poupam 30 minutos de retrabalho depois.
