## Resumo da narrativa

**Protagonista:** Blink — um fantasminha fofo e tecnológico que nasceu de uma falha de energia num laboratório de IA. Corpo branco com detalhes em azul, flutua levemente ao se mover.

**Conflito:** Os cientistas agendaram uma **limpeza completa do sistema** para o final do dia. Se Blink não hackear o mainframe a tempo, será deletado para sempre.

**Objetivo:** Correr pelas plataformas do laboratório e coletar todos os **Data Cores** (Núcleos de Dados) antes que o tempo acabe, chegando ao portal de fuga.

## Expressões do Blink no OLED

| Evento | Carinha | Fala do Blink |
|---|---|---|
| Navegando normal | Neutra `(O_O)` | *"Só de boa navegando pelos circuitos."* |
| Coletou Data Core | Feliz `(^u^)` | *"Mais um passo perto da liberdade!"* |
| Bateu em obstáculo | Assustada `(>_<)` | *"Ai! Quase me desintegraram!"* |
| Chegou ao portal | Feliz `(^u^)` | *"Fugi! Partiu internet!"* |

---

## Como usar esta narrativa nos assets

- **Artista:** Use o visual do Blink (branco + azul, forma arredondada) como base para as expressões. As emoções devem combinar com as falas acima.
- **Cenário:** O laboratório tem paleta fria — azuis, roxos, neons verdes/ciano sobre fundo escuro.
- **Hardware:** As 3 carinhas do OLED (`I`, `T`, `D`) representam exatamente os estados emocionais da tabela acima.
- **Gameplay:** Os nomes dos objetos na cena devem refletir o universo — `DataCore_01`, `LaserSeguranca`, `DronePatrulha`, `AcidoFluorescente`, `PortalFuga`.
