Resumo da proposta

Pitch do jogo

Blink, um fantasminha digital, precisa coletar Data Cores num laboratório sci-fi para hackear o mainframe e escapar antes de ser deletado.

Platformer 2D Unity 6 ESP32 + OLED Evento 1 dia

Diferencial

O Unity envia reações para o ESP32, que devolve comandos de controle. O OLED funciona como extensão emocional do personagem — a plateia vê Blink reagir ao vivo.

Riscos principais

Risco 1

A base funcional está pronta, mas existe espaço para substituir partes do bootstrap por uma montagem 100% manual no layout final da cena.

Risco 2

O bootstrap agora cobre cenário e chão, mas a fase ainda ganha ao ter um layout mais intencional quando aberta no Unity Editor.

Risco 3

Os testes cobrem regras centrais, mas falta validar tudo em runtime no Unity Editor e no hardware real antes da demo.
